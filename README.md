# AI Product Manager Chinese Toolkit 🚀

这是一个围绕 AI 产品学习与实践持续整理的中文资料库。

我会把自己在学习 AI 产品、开源项目、Prompt、Agent、MCP、AI Workflow 和 Vibe Coding 过程中看到的资料、案例和思考沉淀在这里，方便自己长期复盘，也希望能为同样关注 AI 产品实践的人提供一些参考。

这个仓库不只是简单收藏链接，而是希望从产品视角，对 AI 产品、开源项目、Prompt 模板、Skills、Agents、MCP、工具调用、AI 工作流和 Vibe Coding 实践相关内容进行筛选、整理、中文化和场景化改写，让它们更容易被理解、复用和实践。

---

## What This Repository Provides｜这个仓库提供什么 🧭

这个仓库目前主要整理两类内容：

1. **AI Product Notes｜AI 产品笔记**
   整理 Prompt、Skills、Agents、MCP、Tools、Workflows、Methods 等 AI 产品相关主题，并补充产品视角下的理解和观察。

2. **Open-source AI Projects｜开源 AI 项目分析**
   选择具有参考价值的开源 AI 项目，从产品定位、用户场景、核心能力、可复用价值和产品化方向进行分析。

同时，每篇内容都会尽量保留一个统一视角：

> 不只记录“这个工具是什么”，也关注它适合解决什么问题、如何被产品化，以及能为 AI 产品实践提供哪些可复用参考。
